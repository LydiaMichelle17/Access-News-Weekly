# Access News Weekly Demo Site

This is a Vercel-ready static demo for Access News Weekly, a special education current-events curriculum platform.

## What Is Included

- Public marketing website
- Sample Friday Five issue
- Three reading level tabs for each story
- Power Words, Social Passport prompts, comprehension checks, and teacher notes
- Contact and meeting request demo forms
- Print-friendly issue styles
- Brand assets from the Access News Weekly project folder

## Run Locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy To Vercel

1. Create a new Vercel project.
2. Import or upload this folder.
3. Use the default static site settings.

No API keys are required for this demo.

## Future Production Build

The full production version should move this demo into a Next.js, TypeScript, Tailwind, Supabase, Vercel-ready app with:

- Supabase Auth for admin login
- Issues and stories database tables
- Contact and meeting request storage
- Draft, preview, publish, unpublish, and archive workflow
- Optional OpenAI-assisted drafting later

Student accounts, student data collection, payments, LMS integration, and OpenAI API calls are intentionally not included in this demo.
